# 📖 Project Overview

This project analyzes sales performance for **Mwalim Electronics Ltd**, a fictional electronics retailer operating across Kenya.

Using **10,000 simulated sales transactions**, this analysis identifies business trends, evaluates branch performance, measures profitability, and provides actionable recommendations for management.

> **Note:** The dataset used in this project is entirely simulated and was created for educational and portfolio purposes.

---

# 🎯 Business Objective

The management team wanted answers to the following questions:

- Which branch generated the highest revenue?
- Which products sold the most?
- Which product categories are the most profitable?
- How did sales change throughout the year?
- Which payment methods are preferred by customers?
- Which salespersons generated the highest revenue?
- Which customer age groups spend the most?
- How do discounts affect profitability?

# 💡 Key Insights

The analysis revealed several important business findings:

- Nairobi branches generated the highest revenue.
- Smartphones and laptops were the best-performing product categories.
- Revenue increased during peak shopping months.
- Digital payment methods, especially M-Pesa, were widely used.
- Customers aged 26–45 contributed the highest share of revenue.
- Higher discounts generally reduced profit margins.
- Salesperson performance varied across branches.

---

# ✅ Recommendations

Based on the analysis:

- Increase inventory for high-demand products.
- Expand successful strategies from top-performing branches.
- Improve digital marketing targeting key customer segments.
- Optimize discount policies to protect profit margins.
- Introduce employee performance incentives.
- Promote digital payment methods.
- Improve inventory planning before peak sales periods.
